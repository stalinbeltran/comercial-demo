### **Asistente** [2026-06-03T00:20:42]

En DBeaver, el problema es cómo estás ejecutando el script. Hay dos modos:

**El problema:** Si presionas `Ctrl+Enter`, DBeaver ejecuta solo la sentencia donde está el cursor, y puede confundirse con múltiples sentencias.

**Las soluciones:**

1. **Ejecutar como script completo** — selecciona todo el SQL (`Ctrl+A`) y presiona `Ctrl+Alt+X`. Esto ejecuta todas las sentencias en orden.

2. **Verificar el delimitador configurado** — en DBeaver a veces el delimitador no es `;`. Puedes revisarlo en:
   - `Window` → `Preferences` → `Database` → `SQL Editor` → `Statement delimiter`
   - Asegúrate de que esté en `;`

3. **Alternativa: especificar el delimitador explícitamente** al inicio del script:
   ```sql
   DELIMITER ;
   
   CREATE TABLE pais ( ... );
   CREATE TABLE ciudad ( ... );
   ```

La opción más rápida es la primera: selecciona todo y usa `Ctrl+Alt+X` en lugar de `Ctrl+Enter`.
